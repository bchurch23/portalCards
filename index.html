How to add a new player
  1. Duplicate the player-card-template.html file and rename it to the player's name e.g. jake-smith.html
  2. Edit the file — swap in their name, stats, pitches, clip links, headshot
  3. Upload both the updated index.html and new player file to GitHub
  4. Add a row to this index by copying the example <tr> block in the HTML and filling in their details
  5. Copy the link from this dashboard and send it to the player
